# Project Heal

A web application for managing a medical organization’s workflow, including doctors, patients, and appointments.

## Overview

Project Heal is a full-stack website designed to help medical organizations manage doctors, patient information, schedules, and appointments in one place. It provides secure user roles, admin control panels, and real-time data interaction between the frontend and backend.

## Features

- Doctor and patient profile management
- Appointment scheduling and management
- Admin dashboard for overall control
- Role-based access and permissions
- Built with a clean and responsive user interface

## Tech Stack

- Backend: Django, Python, PostgreSQL
- Frontend: HTML, CSS, JavaScript (or React if used)
- Deployment: Docker, Nginx, Linux
- Authentication and API: JWT / Secure auth
